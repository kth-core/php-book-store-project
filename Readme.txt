How to run the Online Book Store  Project
1. Download the  zip file
2. Extract the file and copy store folder
3.Paste inside root directory(for xampp xampp/htdocs)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name store
6. Import store.sql file(given inside the zip package in database folder)
7.Run the script http://localhost/store (frontend) - for user panel
8.Run the script http://localhost/store/admin (frontend) - for admin panel
